# Test Flask App Locally Guide

### 1. Set Flask App Environment Variable
```bash
set FLASK_APP=app
set FLASK_ENV=development
```

### 2. Run Flask Development
```bash
flask run
```

Access it on `http://127.0.0.1:5000`


##### Advanced Local Testing

```bash
flask run --host=0.0.0.0 --port=8000
```
